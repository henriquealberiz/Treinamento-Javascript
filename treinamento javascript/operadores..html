OPERADORES ARITMÉTICOS

 Operadores aritméticos são utilizados para a realizaçao de cálculos simples em Javascript Representam os operadores matemáticos básicos de soma, subtraçao, divisão e multiplicação, além de algumas operaçoes espaciais, como o módulo e incremento/decremento. O Módulo é o resto de uma divisao inteira. Incrementar significa aumentar o valor de algo, e decrementar significa diminuir o valor de algo (uma variavel, por exempo)

Operador Significado
+ Soma
- Subtraçao
* Multiplicação
/ Divisão
% Módulo
++ Pré ou pós incremento
--+ Pré ou pós decremento

OBS: o sinal de + serve para concatenar strings também, logo, é necessário verificar se os tipos envolvidos, sao tipos numéricos.

Abaixo farei um exemplo desses operadores em funcionamento, onde a variavel A irá receber 10 e a variavel B receberá 20.

Código usado acima,, foi o seguinte:
script>
var a=10;
var b=20;
document.write("Soma: "+(a+b)+"br>");
document.write("Subtraçao: "+(a-b)+"/br>");
document.write("Multiplicaçao: "+(a*b)+"/br>");
document.write("Divisao: "+(a/b)+"/br>");
document.write("Módulo: "+(b%a)+"/br>");
document.write("Incremento: "+(++a)+"/br>");
document.write("Decremento: "+(--b)+"/br>");
/ script>
